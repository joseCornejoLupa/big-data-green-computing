# Vault: Big Data + Green Computing

Vault Obsidian dedicado a la investigación sobre eficiencia energética en arquitecturas de Big Data.

## Carpetas

- `papers/` — una nota por paper leído, con resumen, citas clave y relevancia para la investigación
- `ideas/` — hipótesis, preguntas abiertas, conexiones entre papers
- `experimentos/` — diseño de experimentos, variables, resultados preliminares

## Convención de nombres

- Papers: `YYYY_PrimerAutor_TituloCortado.md`
- Ideas: `idea_YYYY-MM-DD_tema.md`
- Experimentos: `exp_NNN_descripcion.md`
